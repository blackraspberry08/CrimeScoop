## Installation
1. Clone the repository
```
mkdir CrimeScoop
cd CrimeScoop/
git clone https://github.com/blackraspberry08/CrimeScoop.git .
```
2. Install packages
```
$ npm install
```

3. Link packages to be sure :)
```
$ react-native link
```

4. Run project
```
$ react-native run-android

or

$ npm run android
```
